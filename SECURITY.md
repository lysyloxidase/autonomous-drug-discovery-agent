# Security Policy

## Supported Versions

Only the current `main` branch is supported during early development.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive reports. Contact the
maintainers privately with a description, reproduction steps, affected version,
and any relevant logs.

## Secrets

Never commit `.env`, API keys, access tokens, private cassettes, or patient data.
The test suite is designed to run without live credentials.
