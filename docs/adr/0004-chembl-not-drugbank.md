# ADR 0004: ChEMBL Instead of DrugBank

## Status

Accepted for later phases

## Context

DrugBank licensing limits redistribution and automated use in open tooling.

## Decision

Use ChEMBL for later molecule and bioactivity workflows unless a deployment has
an explicit DrugBank license.

## Consequences

The project favors open, reproducible data sources and avoids accidental license
violations.
